## 🧠 1. Описание проекта
**Kuro** — это строгий мультимедийный язык программирования, написанный на C++. 

Главная особенность языка — уникальная архитектура **Вилиабл Код (Viliable Code)**:
1. **Глобальное сканирование (Pre-parsing):** Движок сначала строго проверяет спецзнаки и заранее резервирует память под все переменные, циклы и ООП-классы, отключая сбои локали (`class.new`).
2. **Строчное выполнение:** При запуске строки объявлений полностью игнорируются для максимальной скорости работы, а команды выполняются пошагово по нажатию клавиши Enter.

## 💻 2. Синтаксис и Команды
Каждая рабочая строка кода в Kuro обязана завершаться точкой с запятой `;`.

* **Комментарии:** 
  * Одиночный: `^комм^ Твой текст;`
  * Множественный: `^/комм/^ Блок текста ^/комм/^`
* **Создание переменной:** `create.variable имя = 'значение';`
* **Вывод текста в консоль:** `txt.print.console('text: Текст для вывода');`
* **Вывод переменной (вложенный синтаксис):** `Var.print.console(print-FalseText(*имя_переменной*); );`
* **Умная математика:** `mathematics.print.console(math=*+*, ? (15, 5); );` (Поддерживает: `+`, `-`, `*`, `/`)
* **Вложенный рандомайзер:** `random.var.create(имя, ?(int-math, ?(1-70); ); );`
* **Цикл While:** 
  ```text
  While: bool=*имя_переменной*:
  {
     код;
  };
  ```
* **ООП и Защита локали:**
  ```text
  Class name program
  {
     create.class(ИмяКласса)
     {
         class.new[int-0, return=0, class=str];
     };
 };
  ```
* **Графика (Рендер):** `img.pp('путь_к_фото.png');`
* **Аудио-система:** `sound.mp3('путь_к_звуку.mp3');`
* **Очистка экрана:** `clear.console;`
* **Системные предохранители:**
  * `system.error.catch(type=*int-frz*, text: 'Сообщение');`
  * `system.exit.console(code=*0*);`

## 🛠️ 3. Как запустить
1. Перейдите в файл `main.cpp` в данном репозитории и скопируйте код компилятора.
2. Вставьте его в мобильное приложение **CxxDroid** на телефоне или в любой онлайн-компилятор C++.
3. Нажмите кнопку запуска и управляйте выполнением кода Kuro с помощью клавиши **Enter**.
